001 二维数组中的查找

Written by with ♥ on March 13, 2019 in
  1. 直接遍历所有元素，存在相等返回true，不然返回false。

运行时间159ms, 占用内存176316k

public boolean Find(int target, int [][] array) {
  for(int[] array1 : array){
    for(int value :array1){
      if (value == target){
        return true;
      }
    }
  }
  return false;
}
  1. 因为数组每一行都是递增的，可以使用二分查找。

时间 173ms, 占用内存17428k

public boolean Find(int target, int [][] array) {
  for(int[] array1: array){
    if (array1.length == 0 || array1[0] > target || array1[array1.length - 1] < target){
      continue;
    }
    if(array1[0] == target || array1[array1.length - 1] == target){
      return true;
    }
    int minIndex = 0;
    int maxIndex = array1.length - 1;
    int middleIndex;
    int middleValue;
    do {
      middleIndex = (minIndex + maxIndex) / 2;
      middleValue = array1[middleIndex];
      if(middleValue == target){
        return true;
      }else if(middleValue > target){
        maxIndex = middleIndex;
      }else {
        minIndex = middleIndex;
      }
    // 如果是 minIndex < maxIndex 的话, 会导致死循环.
    }while(minIndex < maxIndex - 1);
  }
  return false;
}
