中间操作

中间操作负责处理流中的每个元素。

  1. flatMap

Stream<R> flatMap(Function<T, Stream<R>)将源流中的每个元素转为新的流，然后将这些流中的元素合并为一个新的流。

  List<String> array1 = Arrays.asList("test-101","test-102");
  List<String> array2 = Arrays.asList("test-201","test-202");
  List<List<String>> lists = Arrays.asList(array1, array2);
  List<String> test = lists.stream()
      .flatMap(Collection::stream)
      .collect(Collectors.toList());
  System.out.println(test);
  1. filter

Stream<T> filter<Predicate<? super T>) 过滤出符合匹配条件的元素

  List<String> strings = Arrays.asList(null, "test-001", "test-002");
  System.out.println(strings);
  strings = strings.stream()
      .filter(Objects::nonNull)
      .collect(Collectors.toList());
  System.out.println(strings);```java

  1. collect

<R, A> collect(Collector<? super T, A, R>) 将stream聚合为一个集合（list，set，map）。

  TestModel testModel1 = new TestModel("name-001", "value-001");
  TestModel testModel2 = new TestModel("name-002", "value-002");
  // TestModel testModel3 = new TestModel("name-002", "value-002");
  TestModel testModel3 = new TestModel("name-003", "value-003");
  List<TestModel> list = Arrays.asList(testModel1, testModel2, testModel3);

  # key不可以重复，否则会抛异常 java.lang.IllegalStateException: Duplicate key
  Map<String,String> nameValueMap = list.stream().collect(Collectors.toMap(TestModel::getName, TestModel::getValue));
  System.out.println(nameValueMap);

  Map<String,List<TestModel>> nameListMap = list.stream().collect(Collectors.groupingBy(TestModel::getName));
  System.out.println(nameListMap);

  Map<String,List<String>> nameValueListMap = list.stream()
      .collect(Collectors.groupingBy(TestModel::getName,Collectors.mapping(TestModel::getName, Collectors.toList())));
  System.out.println(nameValueListMap);
